Void vs self-closing elements

Definitions

A void element is a tag that has no closing pair.
Think break <br> and input <input> elements.
A complete list can be found here

A self-closing element does not exist in HTML. In XHTML, or XML, a self-closing element looks like a void tag, but includes a slash / at the end. Like this <br />.
Browsers will ignore the trailing slash.
